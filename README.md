# Anime Website

A modern anime website built using **Next.js** and **Framer Motion** for smooth animations and transitions. The website fetches anime data from the [Jikan API](https://jikan.moe/), providing users with up-to-date information about their favorite anime.

## Features

- **Responsive Design**: Works seamlessly on all devices (desktop, tablet, and mobile).
- **Smooth Animations**: Powered by Framer Motion for a delightful user experience.
- **Anime Details**: Fetch and display detailed information about anime.

## Technologies Used

- **Next.js**: A React framework for server-side rendering and static site generation.
- **Framer Motion**: A production-ready motion library for React.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Jikan API**: An unofficial MyAnimeList API for fetching anime data.
- **TypeScript**: For type-safe development.
